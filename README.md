# daechul-spring-legacy

대철이제철 프로젝트 Spring Legacy 마이그레이션 - JSP 게시판 시스템을 Spring MVC 프레임워크 구조로 전환

## 원본 프로젝트
본 프로젝트는 [daechul-project](https://github.com/every-git/daechul-project)를 Spring Legacy 구조로 마이그레이션하는 작업입니다.

## 마이그레이션 목표
- JSP 기반 게시판 시스템을 Spring MVC 패턴으로 재구성
- MyBatis를 활용한 데이터베이스 연동
- Spring Framework의 DI/IoC 적용
