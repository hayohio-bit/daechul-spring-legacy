## 2. Eclipse 설정하기

### 2.1 새 프로젝트 생성

```
File → New → Maven Project → maven-archetype-webapp 선택
```

```
┌─────────────────────────────────────────────┐
│              Project Facets                  │
├─────────────────────────────────────────────┤
│  ✅ Dynamic Web Module  → 6.0               │
│  ✅ Java                → 21                │
│  ✅ JavaScript          → 1.0               │
└─────────────────────────────────────────────┘

```

## 4. 의존성(라이브러리) 설정 - pom.xml

Maven은 필요한 라이브러리를 자동으로 다운로드해줘요.  
`pom.xml` 파일에 원하는 라이브러리를 적어주면 돼요.

### 4.2 버전 관리 (properties)

버전 번호를 한 곳에서 관리하면 수정이 편해요!

```xml
<properties>
    <java.version>21</java.version>
    <spring.version>6.2.1</spring.version>
    <spring-security.version>6.4.2</spring-security.version>
</properties>
```

### 4.3 의존성 추가 예시

```xml
<!-- 이렇게 적으면 Maven이 자동으로 다운로드해요 -->
<dependency>
    <groupId>org.springframework</groupId>  <!-- 만든 회사/단체 -->
    <artifactId>spring-core</artifactId>     <!-- 라이브러리 이름 -->
    <version>${spring.version}</version>     <!-- 버전 (위에서 정의한 변수 사용) -->
</dependency>
```

## 5. 데이터베이스 설정

### 5.1 MySQL 시작하기

```sql
-- 데이터베이스 생성
CREATE DATABASE daechuldb 
  DEFAULT CHARACTER SET utf8mb4 
  COLLATE utf8mb4_unicode_ci;

-- 사용자 생성 (아이디: daechuluser, 비밀번호: 1234)
CREATE USER 'daechuluser'@'localhost' IDENTIFIED BY '1234';

-- 권한 부여
GRANT ALL PRIVILEGES ON daechuldb.* TO 'daechuluser'@'localhost';
FLUSH PRIVILEGES;
```

### 5.2 연결 정보 (root-context.xml에서 사용)

```
┌─────────────────────────────────────────────────────────────┐
│                    DB 연결 정보                              │
├─────────────────────────────────────────────────────────────┤
│  URL:       jdbc:mysql://localhost:3306/daechuldb           │
│             ?serverTimezone=Asia/Seoul                      │
│             &characterEncoding=UTF-8                        │
│                                                             │
│  Username:  daechuluser                                     │
│  Password:  1234                                            │
│  Driver:    com.mysql.cj.jdbc.Driver                        │
└─────────────────────────────────────────────────────────────┘
```

## 6. 프로젝트 실행하기

### 6.1 Maven 빌드

```bash
# 터미널에서 실행
cd dc-SF
mvn clean install
```

### 6.2 Eclipse에서 실행

```
1. 프로젝트 우클릭
2. Run As → Run on Server
3. Tomcat v10.1 선택
4. Finish

→ 자동으로 브라우저가 열리면서 http://localhost:8080 접속!
```

## 7. Lombok 설정 (추가 설정 필요!)

Lombok은 getter, setter 같은 코드를 자동으로 만들어주는 도구예요.  
Eclipse에 별도로 설치해야 해요:

```
1. lombok.jar 다운로드 (Maven이 받아둔 것 사용 가능)
2. 터미널에서: java -jar lombok.jar
3. Eclipse 설치 경로 지정
4. Install/Update 클릭
5. Eclipse 재시작
```
