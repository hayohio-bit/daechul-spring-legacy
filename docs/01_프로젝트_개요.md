# 🏠 Part 1: 프로젝트 개요

## 1. 이 프로젝트는 무엇인가요?

**대철이제철 게시판**은 회원이 가입하고, 로그인한 후 글을 쓰고, 읽고, 수정하고, 삭제할 수 있는 웹 게시판입니다.

쉽게 말해서:

- 네이버 카페나 학교 공지 게시판 같은 것을 만드는 프로젝트예요
- 회원만 글을 쓸 수 있고, 관리자는 모든 글과 회원을 관리할 수 있어요

## 3. 사용자 역할

| 역할 | 설명 | 할 수 있는 것 |
|------|------|--------------|
| 🔓 비회원 | 로그인하지 않은 사람 | 글 목록/상세보기만 가능 |
| 👤 일반 회원 | 로그인한 사람 | 글 목록/상세/작성/수정/삭제 |
| 👑 관리자 | 특별 권한을 가진 사람 | 모든 기능 + 회원/게시글 관리 |

## 5. 기술 스택을 선택한 이유

| 기술 | 왜 선택했나요? |
|------|---------------|
| Java 21 | 가장 최신이면서 안정적인 버전. 성능도 좋고 새로운 기능도 많아요 |
| Spring Framework | 기업에서 가장 많이 쓰는 프레임워크. 취업에도 유리해요 |
| Spring Security | 로그인/보안을 직접 만들면 실수하기 쉬운데, 이미 검증된 도구를 쓰면 안전해요 |
| MyBatis | SQL을 직접 작성하면서 배울 수 있어요. 복잡한 쿼리도 자유롭게 쓸 수 있어요 |
| MySQL | 무료이면서 성능도 좋고, 배우기 쉬워요 |
